# גיבוי ושליחה ל-GitHub

כשהמשתמש כותב "סיימתי" או "סיימתי לעבוד", בצע את השלבים הבאים:

// turbo-all

1. רענן את ה-PATH של Git:
```powershell
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")
```

2. הצג את השינויים שנעשו:
```powershell
git status
```

3. הוסף את כל השינויים:
```powershell
git add .
```

4. צור commit עם תיאור אוטומטי שכולל תאריך ושעה:
```powershell
git commit -m "עדכון - $(Get-Date -Format 'dd/MM/yyyy HH:mm')"
```

5. שלח ל-GitHub:
```powershell
git push
```

6. הודע למשתמש בעברית שהגיבוי הצליח, כולל רשימת הקבצים ששונו.
